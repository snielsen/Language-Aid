Welcome to the Language Aid SDK!


NOTE: This SDK requires that you have Language Aid installed to function correctly.  

This installer package will install the Language Aid SDK for all users on this machine by installing these components:


  • LAPlugin.h - The header file that plugin modules require.
  • LAUserDefaults.h - The header for a plugin convenience class.
  • Plugin Module Examples - A few shipping examples with complete source and XCode project files.
  • XCode Project Templates - Ready to build XCode project templates for Language Aid plugin modules.
  • Documentation - Instructions and notes on how to build plugin modules.


This installer will install these files into these directories:


/Library/Application Support/Language Aid

/Library/Application Support/Apple/Developer Tools/Project Templates/Language Aid


The SDK will be uninstalled if you ever chose to uninstall Language Aid from the preference pane.
