Welcome to Language Aid!


This installer package will setup Language Aid for all users on this machine by installing these components:


  • Language Aid Application - The actual application
  • Language Aid Preference Pane - Manages configuration and registration
  • Language Aid Contextual Menu Plugin - Allows for use as a contextual menu option
  • Uninstall Script - Removes Language Aid (accessed by the preference pane)
  • Plugin Modules - A few plugin modules written by Aoren Software to start off with:
    • WWWJDIC - Japanese dictionary lookups
    • MDBG - Chinese dictionary lookups
    • Wikipedia - Wikipedia entry lookup
    • Google - Immediate search access
    • Google Translate - Language translation in a variety of languages


This installer will install these files into these directories:


/Library/Application Support/Language Aid

/Library/PreferencePanes

/Library/Contextual Menu Items


If you wish to uninstall this package then click the "Uninstall" button in the preference pane.
